# Unidade I — Fundamentos e instalação

Nesta unidade, o estudante prepara o laboratório, compreende os papéis de um servidor e realiza a instalação inicial de sistemas operacionais voltados à administração de serviços.

## Resultados esperados

- diferenciar servidor físico, máquina virtual, sistema operacional e serviço;
- selecionar recursos coerentes com uma carga de trabalho;
- criar uma topologia de laboratório isolada;
- instalar e atualizar um sistema Linux Server;
- reconhecer equivalências básicas no Windows Server e PowerShell.
